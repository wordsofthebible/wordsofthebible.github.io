About

The entire KJV Bible is arranged in columns from left to right. Type in a search box to highlight where a specific word appears. Hover over the main area to examine passages. Click the color next to a search box to update the word color. Experiment with size and strength to customize the appearance.

Once you have made an interesting view, click the share button and copy a link to share with others.

The source code for this site is available on GitHub. See a problem? Please submit an issue (requires a GitHub account). KJV data was made available by Robert Rouse under a Creative Commons Attribution Share-Alike 4.0 License. See more Bible visualizations at Viz.Bible.
